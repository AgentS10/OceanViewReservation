🔐 Role-Based Access Control

Each role has specific permissions within the reservation system:

👑
Administrator

Full system access — manage users, monitor sessions, CRUD all reservations, view audit log, terminate sessions.

📋
Manager

All reservation operations — create, update, check-in, check-out, cancel. Access admin dashboard and statistics.

🏨
Staff

Front desk operations — create reservations, check-in guests, view bookings, calculate bills. Cannot cancel or check-out.
